## Overview

Amore Jewellery Co provides a sophisticated online shopping experience focused on timeless style and premium craftsmanship. Highlights include:

- Curated luxury items such as butterfly necklaces, zircon bangles, silver bracelets, golden/leather watches, and signature perfumes
- Categories for Women and Men
- Prices ranging from elegant everyday pieces to statement luxury items
- Foundation for advanced features: user authentication (login/register), shopping cart, and product management

This project demonstrates a multi-page static e-commerce structure built with HTML, CSS, and JavaScript.
